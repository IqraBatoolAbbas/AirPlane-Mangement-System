Airplane Management System
Airplane Management System is a comprehensive, web-based platform designed for modern airline operations. It bridges the gap between end-user flight booking and backend operational management, providing a unified dashboard for passengers, ground staff, and administrators.

🚀 Key Features
✈️ For Passengers
Intuitive Booking Flow: A seamless multi-step reservation process for flight selection, passenger details, and add-ons.

Real-time Summary: Dynamic booking summary with automatic cost calculations for meals and luggage.

Informative Policies: Dedicated pages for Luggage Policies and Gourmet Meal menus.

⚙️ For Operations & Admin Team
Unified Dashboard: A central hub showing live key performance indicators (Revenue, Active Flights, Bookings).

Ticket Management: Full control over reservations, check-ins, and booking status.

Live Boards: Real-time tracking of Departures and Arrivals with gate and terminal updates.

Fleet & Crew Management: Track aircraft health, maintenance logs, and manage personnel scheduling.

🛠️ Technology Stack
Frontend: HTML5, CSS3, Bootstrap 5 (Responsive Framework).

UI/UX: Custom design using Manrope & Space Mono typography with a modern dark-themed operational aesthetic.

Icons: Bootstrap Icons.

Logic: JavaScript for multi-step form handling, state management, and real-time UI updates.

📂 Project Structure
index.html – Public Homepage & Booking gateway.

booking.html – Multi-step reservation interface.

dashboard.html – Admin operations overview.

tickets.html – Reservation management center.

fleet.html & crew.html – Administrative asset management.

departures.html & arrivals.html – Live operational boards.

style.css – Centralized custom styling.

💡 How to Run
Clone this repository: git clone [https://github.com/IqraBatoolAbbas/AirPlane-Mangement-System.git](https://github.com/IqraBatoolAbbas/AirPlane-Mangement-System.git)

Open the project folder in any code editor (e.g., VS Code).

Open index.html in your web browser to view the public interface.

Navigate to admin pages via the login/navigation links to explore operations.
